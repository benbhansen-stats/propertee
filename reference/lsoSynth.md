# Synthethic Regression Discontinuity Data

The data for this example were randomly simulated using the **synthpop**
package in R based on data originally collected by Lindo, Sanders, and
Oreopoulos (2010).

## Usage

``` r
lsoSynth
```

## Format

A `data.frame` with 40,403 rows and 11 columns.

- `R`

- `lhsgrade_pct`

- `nextGPA`

- `probation_year1`

- `totcredits_year1`

- `male`

- `loc_campus1`

- `loc_campus2`

- `bpl_north_america`

- `english`

- `age_at_entry`

## Details

See the "Regression Discontinuity StudySpecifications" vignette on the
`propertee` website for more details on the original data, a link to the
code used to generate this synthethic data, and a detailed example.
