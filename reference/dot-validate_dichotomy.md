# (Internal) Validate a dichotomy against other dichotomies found in the call stack

(Internal) Validate a dichotomy against other dichotomies found in the
call stack

## Usage

``` r
.validate_dichotomy(possible_dichotomy)
```

## Arguments

- possible_dichotomy:

  a list or a formula, the former coming from
  [`.find_dichotomies()`](https://benbhansen-stats.github.io/propertee/reference/dot-find_dichotomies.md)

## Value

formula for a dichotomy or NULL
