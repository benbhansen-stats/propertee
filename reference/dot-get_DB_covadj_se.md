# (Internal) Design-based variance for models with covariance adjustment

(Internal) Design-based variance for models with covariance adjustment

## Usage

``` r
.get_DB_covadj_se(x, ...)
```

## Arguments

- x:

  a fitted `teeMod` model

## Value

design-based variance estimate of the main treatment effect estimate

## Details

Calculate design-based variance estimate for `teeMod` models with
covariance adjustment and without absorbed effects
