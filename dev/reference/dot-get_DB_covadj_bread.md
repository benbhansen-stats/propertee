# (Internal) Bread matrix of design-based variance

(Internal) Bread matrix of design-based variance

## Usage

``` r
.get_DB_covadj_bread(x, ...)
```

## Arguments

- x:

  a fitted `teeMod` model

## Value

a list of bread matrices

## Details

Calculate bread matrix for design-based variance estimate for `teeMod`
models with covariance adjustment and without absorbed effects
