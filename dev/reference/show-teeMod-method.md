# Show a `teeMod`

Display information about a `teeMod` object

## Usage

``` r
# S4 method for class 'teeMod'
show(object)
```

## Arguments

- object:

  `teeMod` object, usually a result of a call to
  [`lmitt()`](https://benbhansen-stats.github.io/propertee/dev/reference/lmitt.md).

## Value

`object`, invisibly.
